# Welcome to the basic-E6B-flight-computer repo!

This is our first repository, and we're just CS Majors doing our best to make a silly little flight computer!


The main reasons why we decided to do this project in the first place were to:
1. Understand GitHub
> I think that GitHub (and by extension, Git) can be a bit hard for newcomers (like myself) to even try GitHub. However, trying it out with my wonderful friend Saleema has been nothing but fun! I just hope that future programmers will read this and feel comfortable enough to try!
2. Collaborate and check each other's code
> Piggybacking off the last point, having someone to check your code and help you make sense of C++ is something that already is amazing, and I couldn't ask for more. When I started learning HTML, it was a rule of thumb to always have your code checked by others before you decided to submit the assignment, so I wanted to do just that!
3. Have something under our names before we graduated from college
> Lastly, I think it's very important to learn code just by doing it. I think learning things on your own (depending on the subject) can actually be better than having it taught to you. Most companies want experience over academic credentials, and this is something that I want to do and learn. I'm not saying this will get me into Microsoft (as cool as that would be), but I can say I did something

I know this is something that probably will be overlooked, but it's something that my friend and I are proud of. We are looking forward to seeing how this project of ours turns out!
